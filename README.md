## DNA Tools ##

<img src="https://raw.githubusercontent.com/davejacobs/dna-tools/master/src/client/public/images/dna-gel.jpg" alt="DNA Tools" width="500">

An experimental set of tools for cloning and analyzing DNA sequences.

### Running ###

    npm install
    npm start
